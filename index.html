What is WEBE?

WEBE stands for World Edit: Bedrock Edtion. Founded by the Squawk Squad, this was created to bring the same value of building to Minecraft: Bedrock Edition as Java Edition has with World Edit. World Edit is a powerful tool that allows players to quickly and easily make changes to their Minecraft world, such as building structures or terraforming land. It is a popular tool among players of Minecraft's Java Edition, and WEBE aims to bring that same level of functionality to players of the Bedrock Edition and to eventually exceed it in building.
